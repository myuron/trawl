## What is a trawl ?
trawl is a tool that downloads log objects stored in Amazon S3 and analyzes them cross-sectionally.

## Why trawl ?
By using trawl, you can search efficiently on the terminal.
You can also save it as a file, so you can edit it with your favorite editor or command.
